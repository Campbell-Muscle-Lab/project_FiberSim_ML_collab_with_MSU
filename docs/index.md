# Home

This site hosts files to support the development of a machine-learning based emulator for [FiberSim](https://campbell-muscle-lab.github.io/FiberSim), a computer model of myofilament-level contraction ([PMC8790209](https://pubmed.ncbi.nlm.nih.gov/34932957/)).

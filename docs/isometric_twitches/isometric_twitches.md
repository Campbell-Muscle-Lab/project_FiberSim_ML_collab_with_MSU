# Isometric twitches

Sub-pages describe datasets obtained by simulating isometric twitches with sets of parameter values selected using a Latin hypercube sampling approach.

The technique is based on a [FiberSim demo for unloaded shortening](https://campbell-muscle-lab.github.io/FiberSim/pages/demos/sampling/latin_hypercube/latin_hypercube.html).
